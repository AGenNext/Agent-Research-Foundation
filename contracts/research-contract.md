# Research Contract

## Governing Principle

```text
No strategic decision without evidence.
```

## Research Contract

Each research study should define:

```yaml
research_id: string
question: string
scope: string
sources: []
source_freshness_requirements: []
trust_refs: []
options_considered: []
recommendation: string
assumptions: []
risks: []
limitations: []
decision_refs: []
```

## Research Categories

- academic
- market
- competitor
- product
- technical
- architecture
- licensing
- security
- GTM
- pricing

## Final Rule

Every recommendation should be traceable to current evidence.
